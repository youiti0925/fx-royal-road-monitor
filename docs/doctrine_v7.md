## Procedure Steps

### `fundamental_environment_check` ファンダメンタル環境確認

**HTF と並ぶ最優先 step**. 当日の主要経済指標発表予定 / 中央銀行スタンス / 要人発言予定 / 介入リスク水準(USD/JPY 158超等) / 直近24時間以内の発言・指標 を確認。1つでも該当があれば bias を修正、強い該当なら HOLD 強制。

### `crisis_mode_detection` クライシスモード検出

ATR×3以上の急落/急騰 OR 重要指標発表後60分以内 OR 要人発言キーワード検知時 = クライシスモード。**通常テクニカル判定無効化**、逆張り禁止、トリプル根拠揃いでも見送り推奨。価格事実(ファクトファースト)のみで判断。

### `intervention_zone_check` 介入リスク帯チェック

USD/JPY: 158-162帯 = 介入警戒、162超 = 介入リスク極大。EUR/JPY なども同様の節目水準を確認。介入リスク帯ではロング(円売り)を控え、短期のショート戦略に切替。

### `verbal_intervention_scan` 口先介入スキャン

直近24時間の要人発言を確認し、**エスカレーション語彙**(『断固たる措置』『スタンバイ』『最後の警告』『退避勧告』『大胆な行動』等)が出ているか確認。出ていれば実弾介入リスク急上昇 → ロング戦略全面禁止、ショート優位。

**最優先 step (テクニカル軸)**. W1 / D1 / H4 / H1 のトレンド方向を必ず順に判定。1つでもスキップしたら判定保留(UNKNOWN/HOLD強制)。上位足が明確トレンドの場合、下位足エントリーは原則その方向のみ。

### `environment` 環境認識

h4とd1のトレンド方向を確認。両方向が一致(またはd1がレンジで h4が一方向)のとき以降の手順を進める。判定不能なら HOLD ではなく UNKNOWN を選ぶ(過剰防衛禁止)。

### `htf_dow` 上位足ダウ

h4のダウ状態(HH-HL/LH-LL/range/broken)を確認。判定対象側はこの方向に従う。押し安値(LHL)/戻り高値(LLH)を実体で割らない限り現トレンド継続。

### `htf_lines_check` 上位足ライン位置チェック

**HTF supremacy 原則**. D1/W1 の主要レジサポ・キリ番が直上/直下(ATR×3 以内)にある場合は強制 HOLD。下位足のシグナルだけで判断するのは致命的な罠(『見えない上位足の壁への激突』)。

### `horizontal_levels` 水平線(S/R)

**重要度4フィルターを必須適用**: ①上位足で引ける ②タッチ3点以上 ③キリ番接近 ④賞味期限残あり(ブレイク回数3未満)。レジサポ転換が成立した線は特に重視。

### `trendline` 斜めトレンドライン

押し安値・戻り高値を3点以上連結する斜めライン。最新タッチからの経過時間が短いほど有効。**水平線と同等以上に重要**(トレンド方向・レンジ範囲・動的サポートが可視化される)。

### `wave_pattern` 波形(チャートパターン)

M5/M15上で形成中の反転形(ダブルトップ/ボトム、トリプルトップ/ボトム、H&S/逆三尊、フラッグ、トライアングル等)を認定。形を暗記でなく『今のトレンドがどちらか』を前提にパターン評価。

### `wave_lines` 波形ライン

ネックライン、無効化ライン(stop side)、ターゲットラインを引く。座標は確定したピボットを基準に。

### `fibonacci_zone` フィボナッチ・プライムゾーン

直近トレンドのフィボリトレースメントを引き、価格が **50-61.8% プライムゾーン** に到達したか確認。引き方は上昇=安値→高値方向、下降=高値→安値方向(逆NG)。**点でなくゾーンで待ち構える**。

### `buildup_check` ビルドアップ確認

ブレイク直前に細かな往復・収縮(エネルギー蓄積)があるか。安値切り上げ/高値切り下げの傾向があれば信頼度UP。**単純な突発ブレイクは慎重に**。

### `ma_alignment` MA整合性 (グランビル)

### `indicator_environment` 指標適用環境

### `divergence_check` ダイバージェンス確認

オシレーターと価格の逆行があるか確認。あれば早期警戒シグナルとして記録するが、エントリー根拠としては不足。必ず他の根拠と組み合わせる。

### `stop_zone_estimation` 火薬庫位置推測

### `breakout` ブレイク確定

ネックラインを実体クローズで抜けたか + ビルドアップ前提があるか + 次足の方向継続(フェイク除外)。実体抜けの幅は ATR×0.3 以上を目安。**単純抜けで飛び乗らない**。

### `retest_second_wave` リテスト/第2波

ブレイク後、ネックラインまで戻って反発したか(レジサポ転換成立)。深さ ATR×0.3〜1.0、戻り回数は1〜2回が理想。**第2波が抜けっぱなし追従より高勝率**。

### `confirmation_candle` 確認足

ブレイク方向への中型実体足(ATR×1.0以上)が出現したか。包み足/ピンバー/パワームーブが出れば優先度上昇。逆方向の長ヒゲは確認足とは扱わない。

### `mtf_alignment` MTF整合性

上位足(H1/H4)の方向と下位足(M5/M15)の反転パターンが整合しているか。極意: 上位足のトレンドに逆らわず、下位足の反転ポイントを狙い撃つ。不整合なら HOLD。

### `triple_confluence_count` トリプル根拠カウント

**本 doctrine の最重要 step**. (a)フィボ50-61.8%ゾーン到達 (b)下位足の反転パターン完成 (c)レジサポ転換ライン との3軸が同一価格帯で重なるか確認。3軸=スナイパーエントリー、2軸=候補、1軸以下=見送り。

### `confluence_count` コンフルエンス5軸

(1)ダウ理論 (2)ライン分析 (3)MA(グランビル) (4)オシレーター (5)プライスアクション の5軸でも別途カウント。トリプル根拠と組み合わせて最終判断。

### `entry_price` ENTRY価格

ネックライン回帰時、または確認足のクローズ近辺。具体的な数値を提示。entry_strategy を 'breakout' / 'retest' のどちらかで明示。フィボレベル別ロット調整(38.2%なら1/3、50/61.8%なら本命)。

### `stop_price` STOP価格 (インバリデーション)

**根拠が崩壊する位置に置く**. 反転を根拠にしているなら抜けた時点で根拠崩壊 = 即損切り。entry方式別: ダウ→直近押し安値の下、三尊→右肩の山の少し上、トレンドライン→ラインの明確な外側、水平線→実体抜け位置の外側。

### `target_price` TARGET価格

波形のメジャード・ムーブ、または直近の主要S/R(反対側のサポート/レジスタンスライン到達点)。RR2.0以上を満たすこと。

### `rr_check` RRチェック

(target - entry) / (entry - stop) を計算。**2.0未満なら final_status を HOLD に強制降格**(WAIT_TRIGGER 不可)。

### `event_clear` イベント確認

60分以内に高インパクト指標がないか。あれば WAIT_EVENT_CLEAR、無ければ次へ進める。

## Glossary

### ダブルトップ

高値が2回ほぼ同水準で形成され、間に明確な押し安値がある反転候補形。2山の高値差はATRの0.5倍以内が目安。

### ダブルボトム

安値が2回ほぼ同水準で形成され、間に明確な戻り高値がある反転候補形。2山の安値差はATRの0.5倍以内が目安。

### トリプルトップ / トリプルボトム

ダブルトップ/ボトムの3山版。3度の試行で抜けられない=売買勢力の決着が近い。ダブルより信頼度が高い反転シグナル。

### ヘッドアンドショルダー (三尊)

中央の高値(ヘッド)が左右の高値(ショルダー)より高い、3山の反転候補形。ネックラインは2つの押し安値を結ぶ。エントリー候補: (1)ネックブレイク (2)リターンムーブ(高勝率)。SLは右肩の山の少し上。

### 逆三尊 (Inverse H&S)

三尊を上下反転した買い反転形。エントリー候補と SL placement は三尊と上下対称。

### 上昇フラッグ / 下降フラッグ

強い推進波の後の小幅レンジ。継続パターン。フラッグからのブレイクで元方向継続を狙う。

### アセンディングトライアングル / ディセンディングトライアングル

水平S/Rと斜めTLの組合せ。継続性が高く、水平側のブレイク方向に走りやすい。

### ネックライン

反転形(ダブルトップ等)の押し安値・戻り高値を結ぶ水平または緩斜の線。実体クローズでブレイクされて初めて成立とみなす。

### 波形崩れ

上昇局面なら直近の押し安値(HL)を実体で割った状態。下降局面ならLHを実体で抜けた状態。ヒゲのみの抜けは崩れと見なさない。

### ダウ

上昇=HH(高値更新)とHL(押し安値切り上げ)の連続。下降=LH(戻り高値切り下げ)とLL(安値更新)の連続。崩れた瞬間にダウ転換と判定。

### 押し安値 (Last Higher Low)

上昇トレンドにおける直近のHL。ここを実体で下抜けるまで上昇トレンドは継続する(ダウ理論)。下降トレンドの場合は『戻り高値 (Last Lower High)』。

### フェイラースイング / ノンフェイラースイング

押し安値割れ後すぐ反転して下げるのがフェイラースイング(早いシグナル)。一旦戻ってから本格下降するのがノンフェイラースイング(遅いシグナル、より確実)。

### 確認足 / Confirmation candle

ブレイク方向を支持する次足。中型以上の実体足(ATRの1倍以上)が望ましい。逆方向のヒゲが長い足は確認足とみなさない。

### リターンムーブ / Pullback / 第2波

ネックラインや水平線をブレイクした後、その線まで戻ってきて反発する動き。ATRの0.3〜1.0倍程度の戻りが理想。心理: ブレイクで取り残された逆張り組の損切り買い + 新規順張り買いが同時発生 → 反発が強くなる(レジサポ転換が機能する根拠)。**抜けっぱなしを追わず、第2波(リテスト)が絶好の新規エントリーポイント**。

### 上位足ダウ

判定対象足の上位時間軸(H1, H4, D1, W1)のダウ状態。下位足エントリーは原則として上位足ダウ方向に従う。

### MTF (Multi-Timeframe) 階層

W1 → D1 → H4 → H1 → M15 → M5 の必須スキャン階層。マクロ(上位足)で環境/方向、ミクロ(下位足)でタイミング。**1つでもスキップしたら判定保留**。

### イベント

高インパクト経済指標(NFP, CPI, FOMC等)。発表前後30分は新規エントリー不可、既存ポジは様子見が原則。

### RR

リスクリワード比 = (target - entry) / (entry - stop)。王道は2.0以上を要求。

### ピボット

短期の局所高値・安値。スケールは micro(2-2)、swing(5-5)、major(10-10) の3段階で定義。

### ATR

Average True Range (14期間)。価格変動の標準量。距離・スイング幅・閾値は基本的にATR比で評価する。

### パワームーブ / 大陽線・大陰線

実体が長い、ATR×2 以上の中型〜大型実体足。安値圏での大陽線=強い反転シグナル、高値圏での大陰線=同様。圧倒的な勢いの可視化。

### ピンバー / 下ヒゲ陽線・上ヒゲ陰線

片側のヒゲが実体より明確に長い足。下ヒゲピンバー=一度下落したが強い買いが入った痕跡。サポート上で出ると反発シグナル。

### 包み足 (Engulfing)

前足を完全に飲み込む実体の足。前足赤+今足緑=強気包み足(反転シグナル)。2本を1本に合成すると長い反対ヒゲが現れる(Visual Equation)。

### はらみ足 (Harami)

前足の実体内に収まる小さな足。相場の迷い・三角持ち合い。次に抜けた方向にトレンドが発生しやすい。

### レジサポ転換 / サポレジ転換

サポートだった水平線をブレイクして下抜けると、戻ってきたとき今度はレジスタンスとして機能する(逆も真)。3ステップ: ①天井に2度抑えられる ②天井を力強く突破 ③かつての天井が床に変わる。心理: フェーズ1ブレイク → フェーズ2プルバック → フェーズ3反発。

### MA / 移動平均線

Moving Average. 期間内の平均価格を結んだ線。SMA=単純(滑らか・遅い・保守的) / EMA=指数平滑(直近重視・早い・ダマシ多)。

### グランビルの法則

MA を使ったエントリー8パターン。**王道**は『買いの③: 上向きMAに価格が上から接近して反発』(順張り)。**罠**は『買いの④: 下向きMAから乖離した位置での逆張り』(下降TLに逆らうため非推奨)。MA の傾きと価格方向が一致していることが王道の絶対条件。

### ボリンジャーバンド (BB)

MA±σ。3フェーズ: スクイーズ(エネルギー充電・低ボラ) → エクスパンション(バンド拡大、注意: トレンド発生確定ではない、ダマシあり) → バンドウォーク(価格が上下バンドに沿って進む = トレンド確定 = 順張りエントリー点)。BB単体ではなくダウ理論のトレンド確認と組み合わせる。

### RSI

Relative Strength Index. 0-100 の過熱度。70以上=買われ過ぎ、30以下=売られ過ぎ。**機能環境はレンジ相場のみ**。強トレンドでは70以上に張り付き続けるため、安易な逆張り売りは致命傷になる。

### MACD

短期EMA(12) と 長期EMA(26) の差を MACD線、それを平均化したものをシグナル線、両者の差をヒストグラムで表示。MACD線がゼロを下から上抜け = チャート上で MA がゴールデンクロスするのと同じ意味。最も実用的なシグナルは『MACD線とシグナル線のクロス』。

### ダイバージェンス

価格は高値更新しているが、オシレーター(RSI/MACD)は高値を切り下げている状態(=エネルギー枯渇)。**転換のサインであってエントリーのタイミングではない**。必ずラインブレイク等の他の根拠と組み合わせて使う。

### コンフルエンス / Confluence

複数の独立した根拠が同一価格帯で重なる現象。5軸: (1)ダウ理論 (2)ライン分析 (3)MA(グランビル) (4)オシレーター (5)プライスアクション。重なる数が多いほどエントリーの優位性が高い。

### トリプル根拠 (Triple Confluence)

本 doctrine の核心。**フィボゾーン(50-61.8%) × 下位足反転パターン(ダブルトップ等) × レジサポ転換ライン** の3つが同一価格帯で重なる『ピンポイントの特異点』。プロのスナイパーエントリーの設計図。

### ダブル根拠 (Double Confluence)

2軸が重なる状態。例: 過去のサポート(現在レジ) + フィボ50-61.8%ゾーン。トリプルより劣るが3-4軸で十分採用可。

### インバリデーション (Invalidation)

自分が想定した相場の根拠が完全に崩れる地点。**損切りはここに置く**。entry方式別: ダウ理論→直近押し安値の下、三尊→右肩の山の少し外側、トレンドライン→ラインの明確な外側、水平線→実体抜け位置の外側。RR最低2.0。

### ビルドアップ (Build-up)

ブレイクアウト直前に起きる、細かな売買の交錯。レジスタンス直下で小刻みに値動きして『エネルギーが溜まっている状態』を作り出す。**単純抜けより遥かに高勝率**。4段階: ①攻防 → ②蓄積 → ③決着 → ④爆発。

### ビルドアップ4段階

①攻防(Clash): 抵抗線付近で買いと売りが激しく交錯 / ②蓄積(Accumulation): 小刻みな値動きの中で未決済ポジションが溜まる / ③決着(Resolution): 最終的に一方の勢力が勝つ / ④爆発(Explosion): 均衡破れた瞬間、敗者の損切りも巻き込み一気にレートが走る。

### 上抜けサイン (安値の切り上げ)

レジスタンス付近で反転安値が次第に切り上がる = 買い圧力増大 = 上昇トライアングル的フォーメーション = 上抜け期待度高。

### 下抜けサイン (高値の切り下げ)

サポート付近で反発高値が次第に切り下がる = 売り圧力増大 = 下降トライアングル的 = 下抜け期待度高。

### 火薬庫 (Stop Hunt Zone)

大量の損切り注文が想定される価格帯。直近 swing/major 高安の少し外側に集中する傾向。レートが抜けると強制損切り決済(反対売買)が発動し、一気に価格が加速する『火薬庫』。

### 損切り連鎖加速 (Stop Cascade)

火薬庫を抜けた瞬間の強制損切りが価格をさらに加速させる現象。利確の売り + 損切りの売り + 新規売り = トリプル圧力で大幅な値幅が出る。

### 3層思考 (事実→心理→予測)

オーダー読解の基本フレーム。①事実: 例『1.1100 に大量の売り逆指値』 / ②心理: 含み損抱えた買い勢の損切り設置 / ③予測: 到達すれば一斉発動で下加速。

### 需要供給均衡 (Demand-Supply Equilibrium)

水平線の発生原理。買い勢力が減って売りと一致 = レートが横ばい = 水平線が生まれる原点。

### 反転3要因 (Reversal Triple Motive)

水平線で勢力逆転が起きる3つの独立動機: ①割高/割安の共通認識(例: 160円は買われすぎ) / ②既存ポジションの利確 / ③反転を狙う新規エントリー = ダブル/トリプル圧力で高確率反転。

### キリ番 (Round Number)

心理的節目価格(例: 1.10000, 160.00, 100.00)。大勢の意識が集中するため水平線として機能しやすい。

### ライン賞味期限 (Line Expiration)

同じラインが何度もブレイクされると効力低下。ブレイク回数が3回以上なら信頼度大幅下げ、新しい主要ピボットでライン更新を検討。

### ライン重要度4フィルター

①上位足で引けるか(MTF) ②反転回数は多いか(タッチ頻度) ③心理的節目に近いか(キリ番) ④賞味期限は切れていないか(ブレイク回数)。全水平線をこの4フィルターで篩い分けてから採用。

### フィボナッチ・リトレースメント

トレンド中の押し目/戻り目候補レベルを可視化するツール。直近トレンドの起点(視点=100%)と終点(0%)を結び、間に23.6/38.2/50/61.8/78.6%のレベルを引く。

### 黄金比 61.8%

フィボナッチ数列の隣接2項の比(13/21=0.619, 21/34=0.617...)が収束する値。自然界の法則(ピラミッド、モナリザ、ひまわり等)で繰り返し現れる『心地よい』水準。市場では大勢の注文が集中しやすい最重要リトレースメントレベル。

### プライム・エントリーゾーン (50-61.8%)

本 doctrine の核心エントリー領域。50%(半値戻し)と61.8%(黄金比)を1つの『面』として捉え、価格がこのゾーンに到達してから反転兆しを待つ。**点でなくゾーンで待ち構える**のがパラダイム。

### フィボナッチ引き方ルール

上昇トレンドの場合: 安値=100% / 高値=0% / 必ず安値→高値方向に引く / 逆方向はNG。下降トレンドの場合: 高値=100% / 安値=0% / 必ず高値→安値方向に引く / 逆方向はNG。

### フィボレベル別戦略マトリクス

23.6%=リスク大で実戦使わない / 38.2%=損切りリスク中で1/3ロットの分割買い / 50.0%=半値戻しで本命プライムエリア(リスク小) / 61.8%=黄金比で本命プライムエリア(リスク小) / 78.6%=最終ライン(その先=トレンド失効)。

### ピンポイント・エントリーの罠

フィボのレベル線にぴったり触れて買う/売るのは初心者の罠。パターンA: 線触れて買い→耐えきれず損切り後反転 / パターンB: 線手前で反発→買い逃し。**ぴったり反転は稀**。ゾーン到達 + ローソク足の反転兆し確認後にエントリー。

### ファンダメンタル

経済指標(NFP/CPI/GDP/PMI)、中央銀行政策(FOMC/日銀会合/ECB)、要人発言、地政学リスク、為替介入。**テクニカルがどんなに揃っても一瞬で崩す力**を持つ。常に最上位で意識する。

### 為替介入 (Currency Intervention)

通貨当局(財務省/日銀)による市場直接介入。実弾介入(数千億円規模の売買)と口先介入(発言だけ)がある。USD/JPY 158-162 帯で実弾介入リスク高。介入後の自律反発はトラップ(2段構え狩りの第1段)。

### 口先介入 (Verbal Intervention)

要人発言による事実上の介入。実弾を伴わなくても市場を動かす。エスカレーション語彙: 『注視』(弱) < 『憂慮』 < 『過度な変動は望ましくない』 < 『行き過ぎた動きには断固たる措置』 < 『スタンバイ』 < 『最後の警告』 < 『最後の退避勧告』(強). 強い語彙が出たら実弾介入リスク急上昇。

### 要人発言キーワード

財務省/日銀/FRB高官のエスカレーション語: 『断固』『断固たる措置』『スタンバイ』『行動』『警告』『退避』『過度な変動』『行き過ぎた動き』『注視』『憂慮』『大胆な行動』。これら検知時はテクニカルより優先で警戒。

### 介入の2段構え狩り (2-Stage Intervention Trap)

当局がよく使う手口。Step1: 1段目の急落(5円幅) → Step2: 自律反発で大衆ロングが溜まる(『介入終わった、買い場』と判断) → Step3: 当局の罠(2発目の介入)、ロングが溜まったタイミングで打ち込む → Step4: ロスカット連鎖、当局は少ない資金で効率的にもう1段下落させる。**1回目反発で買う大衆 = 2回目下落の養分**。

### 養分 (Liquidity Bait)

プロ・当局が狙う逆張り大衆のポジション。急落後の自律反発でロング、急騰後の自律調整でショート、を入れる素人がここに含まれる。彼らのストップロスが次の一撃の燃料になる。

### 落ちてくるナイフ (Falling Knife)

急落中の通貨/銘柄。底値判定不能。**カウンタートレード(逆張り買い)は単なるギャンブル**。下落が止まる保証なし、もう一段下がる可能性極大。プロは絶対に掴まない。

### 急落V字回復パターン (Crash V-Recovery)

急落相場の頻発パターン: 急落 → 一時的反発 → 再度下を掘る(本当の底) → 本格的な V字回復。介入に限らず急落全般に出る。**プロの戦術**: 反発で売り(Strategy 1)、2回目の底(直近安値下抜け確認)で買い(Strategy 2)、波を往復で取る。

### ファクトファースト原則 (Fact-First Doctrine)

『介入か?推測か?』の議論は完全に無意味。介入かどうか的中しても1円の利益にもならない。**重要なのは『事実として何が起きているか(価格動向)』と『どう立ち回るか』のみ**。憶測でなく価格事実で判断。

### クライシスモード (Crisis Mode)

ATR×3以上の急落/急騰、または重要指標発表後60分以内、または要人発言キーワード検知時。通常のテクニカル判定が無効化され、特別ルールが適用される(逆張り全面禁止、トリプル根拠でも見送り推奨)。

### 中央銀行スタンス (Central Bank Stance)

ハト派(Dovish, 緩和的) / タカ派(Hawkish, 引締め的) / 中立。FOMC/日銀/ECB の現在スタンスが通貨方向の長期バイアス。スタンスに逆らう取引は短期でも警戒。

### 経済指標サプライズ (Economic Surprise)

発表値が市場予想と乖離した度合い。ポジティブサプライズ(予想超え)/ネガティブサプライズ(予想未達)。サプライズの方向と通貨反応がしばしば一致しない(『材料出尽くし』反転)ため、発表直後60分は新規エントリー禁止が原則。

### 発表後の3パターン

重要指標発表後の典型的価格動作: ①順張り継続(サプライズ方向に走り続ける) ②V字反転(初動と逆方向に戻る、材料出尽くし) ③レンジ収束(初動後に元の水準に戻る、ポジション調整完了)。発表後60分以内はパターン確定まで観察のみ。
