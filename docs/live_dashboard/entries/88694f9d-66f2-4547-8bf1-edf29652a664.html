- **ネックライン**: 反転形(ダブルトップ等)の押し安値・戻り高値を結ぶ水平または緩斜の線。実体クローズでブレイクされて初めて成立とみなす。
- **波形崩れ**: 上昇局面なら直近の押し安値(HL)を実体で割った状態。下降局面ならLHを実体で抜けた状態。ヒゲのみの抜けは崩れと見なさない。
- **ダウ**: 上昇=HH(高値更新)とHL(押し安値切り上げ)の連続。下降=LH(戻り高値切り下げ)とLL(安値更新)の連続。崩れた瞬間にダウ転換と判定。
- **確認足**: ブレイク方向を支持する次足。中型以上の実体足(ATRの1倍以上)が望ましい。逆方向のヒゲが長い足は確認足とみなさない。
- **リターンムーブ**: ネックラインや水平線をブレイクした後、その線まで戻ってきて反発する動き。ATRの0.3〜1.0倍程度の戻りが理想。
- **上位足ダウ**: 判定対象足(M5)の上位時間軸(H1, H4, D1)のダウ状態。下位足エントリーは原則として上位足ダウ方向に従う。
- **イベント**: 高インパクト経済指標(NFP, CPI, FOMC等)。発表前後30分は新規エントリー不可、既存ポジは様子見が原則。
- **RR**: リスクリワード比 = (target - entry) / (entry - stop)。王道は2.0以上を要求。
- **ピボット**: 短期の局所高値・安値。スケールは micro(2-2)、swing(5-5)、major(10-10) の3段階で定義。
- **ATR**: Average True Range (14期間)。価格変動の標準量。距離・スイング幅・閾値は基本的にATR比で評価する。

## 王道14手順 (全手順について判定し procedure_steps[] に出力すること)
1. **環境認識** (environment): h4とd1のトレンド方向を確認。両方向が一致(またはd1がレンジで h4が一方向)のとき以降の手順を進める。
2. **上位足ダウ** (htf_dow): h4のダウ状態(HH-HL/LH-LL/range/broken)を確認。判定対象側はこの方向に従う。
3. **水平線(S/R)** (horizontal_levels): 直近24時間〜1週間で複数回機能した水平価格帯を特定。touchが3点以上なら強い、2点以下は弱い。
4. **トレンドライン** (trendline): 上位足の主要トレンドラインを特定。3点以上の接触があり、最新タッチからの経過時間が短いものほど有効。
5. **波形** (wave_pattern): M5上で形成中の反転形(ダブルトップ/ボトム、H&S、トライアングル等)を認定。形成中なら未確定として扱う。
6. **波形ライン** (wave_lines): ネックライン、無効化ライン(stop side)、ターゲットラインを引く。座標は確定したピボットを基準に。
7. **ブレイク確定** (breakout): ネックラインを実体クローズで抜けたか。ヒゲのみの抜けは未確定。実体抜けの幅はATR×0.3以上を目安。
8. **リターンムーブ確定** (retest): ブレイク後、ネックラインまで戻って反発したか。深さATR×0.3〜1.0、戻り回数は1〜2回が理想。
9. **確認足** (confirmation_candle): ブレイク方向への中型実体足(ATR×1.0以上)が出現したか。逆方向の長ヒゲは確認足とは扱わない。
10. **ENTRY価格** (entry_price): ネックライン回帰時、または確認足のクローズ近辺。具体的な数値を提示。
11. **STOP価格** (stop_price): 波形が無効化される位置(直近対極ピボットの外側)。entry からの距離はATR×1.0〜2.5。
12. **TARGET価格** (target_price): 波形のメジャード・ムーブ、または直近の主要S/R。RR2.0以上を満たすこと。
13. **RRチェック** (rr_check): (target - entry) / (entry - stop) を計算。2.0未満なら見送り、2.0以上で合格。
14. **イベント確認** (event_clear): 60分以内に高インパクト指標がないか。あれば WAIT_EVENT_CLEAR、無ければ次へ進める。

## 参考例 (王道流儀の判定例)

### fs1: EURUSD M5 - ダブルトップ完成 + ネック割れ後のリターンムーブ待ち
状況: 1.10500付近で2山が形成され、押し安値1.10300がネックライン。M5実体で1.10280までクローズしブレイク確定。h4ダウは下降中、d1はレンジ。
期待判定: side=SELL, final_status=WAIT_RETEST
  → ダブルトップ + ネック割れ確定。ネックラインへのリターン狙いで待機。

### fs2: EURUSD M5 - 上位足と方向不一致、見送り
状況: M5 で下降ダブルトップ形成中だが、h4とd1がともに強い上昇トレンド。M5の反転は上位足に対する逆張りに当たる。
期待判定: side=NEUTRAL, final_status=HOLD
  → 上位足と方向不一致のため王道対象外。

### fs3: EURUSD M5 - 60分以内にFOMC、待機
状況: ダブルボトム + ネックライン上抜けの設定だが、25分後にFOMC声明発表予定。
期待判定: side=BUY, final_status=WAIT_EVENT_CLEAR
  → 波形は揃っているが直近にFOMC。イベント通過後に再評価。

### fs4: EURUSD M5 - ブレイク未達、波形監視継続
状況: ダブルトップらしい2山は出ているが、ネックラインへの実体クローズ抜けがまだない。
期待判定: side=SELL, final_status=WAIT_BREAKOUT
  → 波形は揃ったがブレイク未確定。ネック実体抜けを待つ。

### fs5: EURUSD M5 - ノイズ、設定なし
状況: ATRが極端に低くレンジ収縮。ピボットが micro スケールしかなく、明確な反転形・ブレイク候補がない。
期待判定: side=NEUTRAL, final_status=SUPPRESSED
  → 明確な王道セットアップ不在。観察対象なし。
